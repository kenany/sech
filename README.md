# sech

Calculate the hyperbolic secant of a number.

## Example

``` javascript
var sech = require('sech');

sech(Math.PI);
// => 0.08626673833405443
```

## Installation

``` bash
$ npm install sech
```

## API

``` javascript
var sech = require('sech');
```

### `sech(x)`

Given a _Number_ `x`, returns the hyperbolic secant of `x` as a _Number_.
